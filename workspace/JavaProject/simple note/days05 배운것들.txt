 - else

if, else 쓰는건 어렵지 않다! ( ) 안에 조건을 쓰는게 복잡해지면 이게 어려워지는 것!


ControlOpIf03

 -else if( )
 else를 사용하는건 좋은데 결과의 경우의 수가 많아지면 코드가 복잡해지고 난장판이 된다.
 이 떄, else 대신 else if()를 적고, ( ) 안에 조건을 쓰면 조건이 true가 된다면 해당 중괄호 { } 안의 내용을 실행한다. 아니라면 다음 조건으로 넘긴다.
